**Bindings**

You run the code below from the file editor.

```python
usa_gold = 46
uk_gold = 27
romania_gold = 1

total_gold = usa_gold + uk_gold + romania_gold
print(total_gold)

romania_gold += 1
print(total_gold)
```

What’s printed?


1. 74 then 74
2. 74 then 75
3. 74
4. 75
